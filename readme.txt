ISTRUZIONI PER L'ESECUZIONE DELLA BATTAGLIA NAVALE

Comandi per la battaglia navale (il log viene salvato in log.txt):
./battaglia_navale pc -> avvia una partita tra giocatore umano e computer
./battaglia_navale cc -> avvia una partita tra due computer

Comandi per il replay 
./replay v [nome_file_log] -> stampa a video il replay del file di log indicato
./replay f [nome_file_log] [nome_file_output_replay] -> scrive sul file indicato il replay del file di log

Vengono inclusi i file:
	computer_vs_computer.txt -> contiene il log di una partita tra due computer
	human_vs_computer.txt  -> contiene il log di una partita tra computer e umano
